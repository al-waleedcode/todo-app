# 📝 Interactive To-Do List App

A modern, responsive To-Do List application built with **Vanilla JavaScript** and styled using **Tailwind CSS**. This project demonstrates fundamental web development concepts including DOM manipulation and persistent data storage.

## 🚀 Key Features
* **CRUD Operations:** Add, view, edit, and delete tasks seamlessly.
* **Local Storage Integration:** Your tasks are saved locally in the browser, so they persist even after refreshing the page.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using Tailwind CSS.
* **User-Friendly Interface:** Clean UI with intuitive interactions and real-time updates.

## 🛠️ Tech Stack
* **HTML5** - Structure and layout.
* **Tailwind CSS** - Modern utility-first styling.
* **JavaScript (ES6+)** - Core logic and functionality.
* **LocalStorage API** - For data persistence.

## 📖 What I Learned
Through this project, I strengthened my understanding of:
1.  **State Management:** Handling arrays of objects to represent application data.
2.  **Event Handling:** Managing user interactions (clicks, inputs, and form submissions).
3.  **Data Persistence:** Using `localStorage.setItem` and `JSON.parse` to store and retrieve data.
